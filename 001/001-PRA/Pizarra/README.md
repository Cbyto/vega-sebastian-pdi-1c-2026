# 🎨 Pizarra Tipo Paint — py5

Aplicación de dibujo estilo Paint desarrollada en Python con la librería **py5**, como trabajo práctico de la materia **Laboratorio de Procesamiento de Imágenes Digitales**.

---

## ✨ Funcionalidades

- ✏️ **Lápiz** — dibujo libre a mano alzada
- 🧹 **Goma** — borrador con tamaño ajustable
- ⬜ **Cuadrado / Rectángulo** — click y arrastrá para definir el tamaño
- ⭕ **Círculo** — proporciones 1:1 automáticas
- 〇 **Elipse** — ancho y alto independientes
- 🎨 **Paleta de 12 colores** seleccionables
- 🔲 Modo **Contorno** o **Relleno** para las figuras
- **Scroll del mouse** para cambiar el tamaño del pincel / goma
- Tecla **`C`** para limpiar el canvas
- Agregado: **Ventana de Ayuda**

---

## 🖥️ Requisitos

- Python 3.8 o superior
- Java JDK 17 o superior → [Descargar aquí](https://www.oracle.com/latam/java/technologies/downloads/)
- Librería py5

## ▶️ Uso

```bash
python pizarra_paint.py
```

Se abrirá una ventana de **800×600 px** con la barra de herramientas en la parte superior.
